Sobre o Gtime auto-completer

Trabalhar também é lidar com burocracias. No Brasil, o horário estipulado de trabalho no regime CLT é de 8 horas diárias. Junto a isso, muitas empresas empregam diferentes sistemas de controle de carga horária. A Google é uma delas.

O aplicativo Gtime tem o papel específico de permitir que os contribuidores definam seus horários de trabalho diariamente. Entretanto, esse processo acaba sendo repetitivo e, muitas vezes, contra-produtivo.

O objetivo deste projeto é automatizar o preenchimento do Gtime, com o usuário definindo, para cada dia da semana, qual seu formato de contribuição.

Especificações

  • O software deve atender tanto FTE (full-time employees) quanto estagiários.
  • A aplicação idealmente deve ser uma extensão de navegador, sem a necessidade de instalação local.
  • Permitir a definição de um "dia típico" de trabalho para cada dia da semana.
  • Possibilidade de editar o quadro de horas de um dia específico, se necessário.

Tecnologias

  • Javascript
  • HTML + CSS
  • Tecnologias de armazenamento no navegador
